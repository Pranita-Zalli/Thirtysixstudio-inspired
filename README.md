# 🎨 Thirtysixstudio | React + GSAP

A modern creative agency-inspired web experience built with **React**, **GSAP**, and **Canvas-based interactions**.

Inspired by the design language and motion principles of Thirtysixstudio, this project explores creative frontend development through immersive interactions, animated typography, smooth transitions, and engaging user experiences.

The goal was not to create an exact replica, but to study and implement modern web design techniques while adding my own frontend development approach and animation workflows.

## ⚙️ Installation & Setup

### Clone Repository

```bash
git clone https://github.com/Pranita-Zalli/Thirtysixstudio-inspired.git
```

### Navigate to Project

```bash
cd Thirtysixstudio-inspired
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```
